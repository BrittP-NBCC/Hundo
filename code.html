The Code

                        
                            //shorthand for getting elements
                            let $ = function(id) { 
                                return document.getElementById(id); 
                            };
                            
                            //get the values from the interface
                            //starts or controller function
                            function getValues(){
                                //get values from the page
                                let startValue = $('startValue').value;
                                let endValue = $('endValue').value;
                            
                                //attempt to parse values into ints
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);
                            
                                //validation that input is an int
                                if(Number.isInteger(startValue) && Number.isInteger(endValue)){
                                    //call generateNumbers
                                    let numbers = generateNumbers(startValue, endValue);
                                    
                                    //call displayNumbers
                                    displayNumbers(numbers);
                                }
                                else alert("You must enter a number.")
                            }
                            
                            //generate numbers from startvalue to the endvalue
                            //logic function(s)
                            function generateNumbers(sValue, eValue){
                            
                                let numbers = [];
                            
                                //we want to get all numbers from start to end
                                for(let index = sValue; index <= eValue; index++){
                                    //this will execute in a loop until index = eValue
                                    numbers.push(index);
                                } 
                                return numbers;
                            }
                            
                            //display numbers with even numbers in bold
                            //display or view function
                            function displayNumbers(numbers){
                            
                                let templateRows = "";
                            
                                for (let index = 0; index < numbers.length; index++) {
                            
                                    let className = "even";
                                    let number = numbers[index];
                            
                                    //checking to see if the remainder of the number is equal to 0. 
                                    if(number % 2 == 0){
                                        className = "even";
                                    }
                                    else {
                                        className = "odd";
                                    }
                                    
                                    //creates rows in table
                                    templateRows += `<tr><td class="${className}" >${number}</td></tr>
                                }
                            
                                $('results').innerHTML = templateRows;
                            }

The Code is structured in four functions to facilitate use of the Model View Controller (MVC).

$

This function will save you time allowing you to pass in the id you want to target with document.getElementById("") and it will return it within the quotes allowing for shorthand down the line

getValues

The getValues function starts by getting the values from the inputs on the app.html page. You can see we use the '$' function and pass in the ids of the specific inputs that we are targeting.
Once we have the inputs stored as values, we then check to see if the values can be stored as integers. If the value cannot be converted to an integer, the value will be changed to 'NaN' here (not a number).
Once we have attempted to parse our data, we must validate that the inputs are integers because we do not want to store multiple data types into our numbers array.
Number.isInteger will return 'true' so long as the value passed in is an int. If BOTH the values return true, then we will call our generateNumbers and displayNumbers functions and pass in the validated data we collected. If either return false, we will display an alert notifying the user to enter valid data. It is also important to note that this is the function that is called on our buttons "click" event.

generateNumbers

The generateNumbers function takes in the validated data as parameters and returns a list of all the numbers between the two values the user selected. First, we initialize our array by setting it = []. We then use a for loop to gather each number one at a time between our two values: sValue and eValue. We create a variable named index and set it to the starting value sValue. The loop will continue on so long as the index is less than or equal to the end value we passed in. we then increment the index by 1 using index++ so we dont create an infinite loop. Every time we get inside the loop code block, we are adding the index to our numbers array using numbers.push(index);. Once we reach the final number, we will return our complete array which is then called in getValues.

displayNumbers

In this function, we will take in an array as a parameter and loop through each number in our numbers array and create a string of HTML code that we will insert into our table. We start by declaring templateRows as a string outside of our loops code block to give it the correct scope. Then, we will create a for loop looping through the numbers array we passed in that will continue until we reach the .length of the array. For every number within our numbers array, we will check to see if it is an even number through use of the modulus operator; which returns the remainder for division by 2, and change a variable called className accordingly. Every time we loop through, we will be concatinating to our templateRows string the className and number. After we have completed our string, we exit the loop and target the inside of the results id with .innerHTML and form our table.
